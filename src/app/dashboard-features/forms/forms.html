Verbose Form
Este formulario maneja un tipado mas estricto. Implementa una interfaz para los FormControls.
Deja que le componente maneje toda la logica del formulario y retorna un Form Value Type
El submit del form retorma una respuesta que esta tipada en la interfaz 
          
            export interface FormValueType {
              name: FormControl< string >;
              email: FormControl< string >;
            }
